# TS-NG_articles

## What does it have?

- Angular CLI 10.x template
- SCSS for styling
- Font Awesome and Bootstrap 4.x (no javascript, just styles)
- Navigation/layout
- Theming
- Linting/formatting - prettier, tslint, eslint, stylelint
- Circle CI 2 integration
